# JWT Local Variable
A way to store JSON Web Token on Local Variable to make it more secure from an XSS and CSRF attack.

Developed using `node v16.13.1`

>Start: __30 September 2021__ | End: __19 June 2022__

## Getting Started
```
yarn install
```
### Development
```
yarn dev
```
### Build
```
yarn build
```
### Test
```
yarn test:watch
```
